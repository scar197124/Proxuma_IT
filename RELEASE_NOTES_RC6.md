# PROXUMA IT UI 1.0 RC-6 — Fixed Summary & Scroll Discovery

## Release blockers addressed

- Captain Summary now holds one consistent desktop/tablet height after a scan.
- Risk, Confidence, Action, and State metrics are compacted into a stable four-column lane.
- Longer Captain Summary text scrolls internally rather than growing the dashboard.
- Investigation now displays a visible “More investigation details” cue when content extends below the viewport.
- The cue disappears automatically when all content is visible or the user reaches the bottom.
- A visible styled scrollbar and keyboard-focusable investigation reader improve discoverability and accessibility.
- Mobile keeps a natural auto-height summary and a two-column metric layout.

## Regression guard

RC-5 risk colors and RC-4 scan-render synchronization remain preserved.

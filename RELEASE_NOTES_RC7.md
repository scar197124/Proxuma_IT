# PROXUMA IT UI 1.0 RC-7

## Hard-Locked Summary + Focus Restoration

- Captain Summary is fixed at a strict height on desktop, tablet, and mobile.
- Risk, Confidence, Action, and State are compacted into a stable 2x2 mobile grid.
- Long action or explanation text scrolls inside the summary instead of enlarging it.
- Investigation case/status text has its own compact readable scroll area.
- Exiting Investigation Focus now restores and re-renders the selected view.
- Investigation content can no longer remain hidden after Focus exit.
- RC-4 scan relay repair, RC-5 risk colors, and automatic analysis updates remain intact.

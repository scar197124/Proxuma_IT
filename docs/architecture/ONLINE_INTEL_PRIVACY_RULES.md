# Online Intel Privacy Rules — v3.28.0

Proxuma IT's online layer must preserve these rules:

1. Offline-first remains the default.
2. Consent must be explicit and revocable.
3. User-opened external links remain allowed.
4. Automatic provider lookups require a serverless privacy boundary.
5. Browser code must not contain provider secrets.
6. Provider calls must use minimum necessary target data.
7. Results must be summarized for users in plain language.
8. If online checks fail, Proxuma must fall back to the local report.
9. Online findings must not overwrite the local verdict silently.
10. No telemetry, hidden analytics, or background submission.

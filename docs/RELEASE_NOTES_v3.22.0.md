# Proxuma IT v3.22.0 — QR / Manual Payload UX Upgrade

This release adds a clearer offline paste lane for decoded QR payloads, copied SMS/email content, and raw suspicious text.

## Added
- Manual Payload Helper under the scanner privacy strip.
- Load to Target action for reviewing pasted payloads before scanning.
- Scan Payload action for immediate local analysis.
- Clear Payload action with privacy-safe local status text.

## Preserved
- Offline-first behavior.
- No telemetry.
- No hidden lookups.
- No active provider/API calls.
- Local scan history from v3.21.0.
- Case packet TXT/JSON export from v3.20.0.

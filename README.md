# Proxuma IT v3.54.0 — Public Release Candidate

A compact, local-first link, message, domain, and QR risk-analysis workspace.

## Release-candidate improvements

- Unified Review → Decide → Act investigation workflow
- Progressive interface that reveals tools after a scan
- Reliable Overview, Deep Analysis, Evidence, and Case File views
- Saved cases, rescan comparison, case management, and local report export
- Live domain intelligence with clear source and freshness status
- Keyboard navigation for tab groups
- Dialog focus containment and Escape-to-close support
- Strong visible focus states and reduced-motion support
- Empty-scan validation and user-visible runtime error recovery
- Accessible live risk-meter values

## Deploy

Upload the contents of this folder to the root of the GitHub repository. GitHub Pages can serve `index.html` directly. The optional `api/` and `vercel.json` files support the live RDAP bridge when deployed through Vercel.

No build command is required.

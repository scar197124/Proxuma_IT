# Proxuma IT v3.50.0

Simplified deployment package.

## Deploy
- GitHub Pages: upload all files and folders in this directory to the repository root.
- Vercel: import the repository; `vercel.json` and `api/proxuma-rdap.js` provide the optional live RDAP bridge.

Do not change the `assets`, `api`, or file paths unless you also update `index.html` and `404.html`.

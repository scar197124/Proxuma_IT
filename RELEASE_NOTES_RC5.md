# PROXUMA IT UI 1.0 RC-5 — Risk Color Intelligence

## What changed
- The completed scan score now selects one live risk tone.
- Analysis Views, Captain Summary, Analysis Results, Investigation, Supporting Evidence, and Findings Into Action share that tone.
- The Analysis Views status now displays the active band and score.
- Active analysis tabs, score ring, heat bar, borders, and status badges respond to the scan.
- Both dark and light themes preserve readable contrast.

## Risk tones
- 0–9: Minimal Risk — green
- 10–19: Low Risk — bronze
- 20–39: Needs Review — amber
- 40–59: Elevated — orange
- 60–79: High Risk — red
- 80–100: Critical — magenta-red

Run each example target and confirm the Analysis Views accent changes immediately with the score.
